确知信号的类型

定义

  • 确知信号： 可用一个确定时间函数表示，取值在任意时间都是可以确定和可预知的信号

按照周期性分类

周期信号

  • 定义： 每隔一定时间间隔按相同规律重复且无始无终的信号
    $s(t) = s(t + T_0),\quad -\infty < t < \infty$
    满足上式的最小正数 $T_0$ 称为信号的基波周期
  • 分类：
    类型名称 类型定义 例子
    简单周期信号 只包含一个频率分量的周期信号 $x(t) = A\sin(2\pi f t)$
    复杂周期信号 由多个频率分量（基波及其谐波）组成的周期信号 $x(t) = \sin(t) + 0.5\sin(2t)$

非周期信号

  • 定义： 不满足周期信号定义的信号
  • 分类：
    类型名称 类型定义 例子
    准周期信号 由多个周期信号合成，但各个信号的频率不成公倍数 $x(t) = \sin(t) + \sin(\sqrt{2}t)$
    瞬态信号 持续时间有限的信号 $x(t) = e^{-\beta t} A\sin(2\pi f t)$

按照信号能量是否有限区分

  • 背景：

    • 在通信理论中，把功率定义为电流在单位电阻($1欧姆$)消耗的功率(归一化功率)。
      因此电流的平方和电压的平方都等于功率
      $P=V^2/R=I^2R=V^2=I^2$

    • 用$s(t)$表示信号电流或电压的时间波形，则$s(t)^2$代表瞬时功率

      • 信号能量表示为：
        $E = \int_{-\infty}^{+\infty} s^2(t) dt$
      • 信号功率表示为：
        $P = \displaystyle \lim_{T \to \infty} \frac{1}{T} \int_{-T/2}^{+T/2} s^2(t) dt$

能量信号

  • 定义： 能量有限，功率为零的信号
    • 数学表达式：$0 < E < \infty, P -> 0$，则s(t)为能量(有限)信号。
  • 例子： 单个矩形脉冲

功率信号

  • 定义： 能量无限，功率有限的信号
    • 数学表达式：$0 < P < \infty, E -> \infty$，则s(t)为功率(有限)信号。
  • 例子： 直流信号、周期信号和随机信号

周期信号的能量功率特性
对于任意周期为$T_0$的周期信号$x(t)$，其能量$E$和功率$P$分别为：

  • 能量：$E = \displaystyle \lim_{T \to \infty} \int_{-T/2}^{+T/2} |x(t)|^2 dt = \lim_{n \to \infty} \int_{-nT_0/2}^{nT_0/2} |x(t)|^2 dt = \infty$
  • 功率：$P = \displaystyle \lim_{T \to \infty} \frac{1}{T} \int_{-T/2}^{+T/2 } |x(t)|^2 dt = \frac{1}{nT_0} \int_{-nT_0/2}^{nT_0/2 } |x(t)|^2 dt = P_0$
  • 结论： 所以任意周期信号都是功率信号，且其功率等于在一个周期内的平均功率

确知信号的频域性质

功率信号的频谱

周期性功率信号的频谱

  • 定义：
    对于周期为 $T_0$ 的功率信号 $s(t)$，可以用指数型傅里叶级数展开为：
    $s(t) = \displaystyle \sum_{n=-\infty}^{\infty} C_n e^{j 2\pi n t/T_0}$
    其中,傅里叶系数$C_n$为：
    $C_n = \displaystyle \frac{1}{T_0} \int_{-t_0/2}^{t_0/2} s(t) e^{-j 2\pi n f_0 t} dt$
    $C_n$称为信号的频谱，它反映了信号中各次谐波的幅度和相位特性。
    其中，$f_0 = 1/T_0$为信号的基波频率，$nf_0(n = 0, \pm 1, \pm 2, \ldots)$称为信号的n次谐波信号

    对于周期性功率信号来说，频谱$C_n$是离散的，只有在$f_0$的整数倍上有取值
    n=0时，有
    $C_0 = \displaystyle \frac{1}{T_0} \int_{-T_0/2}^{T_0/2} s(t) dt$ 称为信号的直流分量

频谱的表示

  • 幅度谱与相位谱
    幅度与相位在频谱上体现为 $C_n=|C_n|e^{j \theta_n}$
    • $C_n$的模$|C_n|$称为幅度谱，反映各次谐波的幅值特性
    • $C_n$的幅角$\theta_n$称为相位谱，反映各次谐波的相位特性

频谱的性质

  • 对于物理可实现的实信号，有
    $C_n=\displaystyle \frac{1}{T_0} \int_{-T_0/2}^{T_0/2} s(t) e^{+j 2\pi n f_0 t}dt=[\frac{1}{T_0} \int_{-T_0/2}^{T_0/2} s(t) e^{+j 2\pi n f_0 t}dt]=\overline{C_n}$

    • 含义： 正负频率分量间存在复数共轭关系

    • 与应的幅度谱为偶函数，相位谱为奇函数

    • 将上式代入傅里叶级数表达式中，可得
      $s(t) =\displaystyle C_0 + 2\sum_{n=1}^{\infty} |C_n| \cos(2\pi n f_0 t + \theta_n)$

      其中
      $|C_n|=\frac{1}{2} \sqrt{A_n^2 + B_n^2}$ 乘坐各次谐波的振幅
      $\theta_n=\tan^{-1}(\frac{B_n}{A_n}) $

      $A_n$ 为n次谐波的实分量，$B_n$ 为n次谐波的虚分量

    $|C_n|$与$\theta_n$都称为单边谱，而$C_n$称为双边谱

    • 性质：
      • 若$s(t)$是实偶信号，则$C_n$为实信号
      • 若$s(t)$不是偶信号，则$C_n$为复信号

能量信号的频谱密度

频谱密度的定义

  • 能量信号s(t)的傅里叶变换： $S(f)=\displaystyle \int_{-\infty}^{\infty} s(t) e^{-j 2\pi f t}dt$
  • S(f)逆傅里叶变换生成原信号： $s(t)=\displaystyle \int_{- \infty}^{\infty}S(f) e^{j 2\pi ft;
    }$

频谱密度$S(f)$与$C_n$的区别

实能量信号的$S(f)$与实功率信号$C_n$的共同特性

  • 如上文讲到，负频谱与正频谱的模偶对称，相位奇堆对称，即复数共轭。
    因为：
    $\displaystyle \int_{-\infty}^{\infty} s(t) e^{-j 2\pi f t}dt=\overline{[\int_{-\infty}^{\infty} s(t) e^{+ j 2\pi f t}dt]}, S(f)=\overline{S(-f)}$
